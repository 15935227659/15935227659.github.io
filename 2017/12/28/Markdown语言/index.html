Markdown语言怎么用

发表于 2017-12-28 | 分类于 编写

1.标题

  • 标题加1个#号是一级标题，加2个#是二级标题依此类推

  • # 一级标题

  • ## 二级标题

2.斜体和粗体

  • 将需要设置为斜体的文字两端使用1个“*”或者“_”夹起来 例如:_88_ 显示为 88
  • 将需要设置为粗体的文字两端使用2个“_”夹起来 例如: __粗体__ 显示为 粗体

3.引用

  • 行首使用 > 加上一个空格表示引用一个段落，可嵌套
    例:> 引用 >>我再引用

    引用

    我再引用

4.分割线

  • 在一行连续使用三个或以上_或*:
    例：-–

5.代码区块

  • 使用 三个“```”来展示代码片段

6.链接和图片

  1. 普通链接 例如:[百度](https://www.baidu.com)
    百度
  2. 图像链接 例如:![](https://assets.servedby-buysellads.com/p/manage/asset/id/38523)

7.无序列表和有序列表

  1. 在文字开头添加(“,”“ +”, 或 “-”)实现无序列表。但是要注意在(“,”“ +”, 或 “-”)和文字之间需要添加空格

    • 这是无序列表
    • 这是无序列表

  2. 使用数字后面跟上句号

    1. 这是有序列表
    2. 这是有序列表
